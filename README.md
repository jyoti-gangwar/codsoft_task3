# Contact Book
## Description
A GUI-based Contact Book application developed using Python and Tkinter. It allows users to store, search, update, and delete contact details. All contacts are saved in a JSON file for future use.

## Features
- Add New Contact
- View Contact List
- Search Contact by Name or Phone Number
- Update Contact Details
- Delete Contact
- Store Contacts using JSON
- Simple and User-Friendly GUI

## Technologies Used
- Python
- Tkinter
- JSON
